# RSA Algorithm (Educational Project)

## Description
This project demonstrates the RSA encryption and decryption algorithm using Python.

## Features
- Key Generation
- Encryption
- Decryption
- Menu-driven CLI

## Warning
This project uses small prime numbers and is NOT secure for real-world use.

## How to Run
```bash
python rsa.py
```

## Example
Enter message: hello
Encrypted: [1234, 5678, ...]
Decrypted: hello
